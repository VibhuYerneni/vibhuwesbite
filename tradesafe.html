TradeSafe

AI-powered mobile application for automated risk management

Overview

TradeSafe is an intelligent mobile application designed to assist investors with automated risk management, intelligent stop-loss & profit-taking systems, and AI-driven market insights.

Key Features

  • Automated Risk Management: Intelligent stop-loss & profit-taking system
  • Schwab API Integration: Direct brokerage account connectivity
  • Options Trading Focus: High-volume options analysis & alerts
  • AI-Powered Insights: Market analysis & trade recommendations
  • Portfolio Protection: Automatic trailing stops & position monitoring
  • Real-Time Data: Live market feeds via Polygon API
  • Secure Authentication: Face ID + OAuth 2.0 token management
  • Push Notifications: Trade alerts & market opportunities

Technical Highlights

Developed and trained a custom machine learning model that outperformed baseline AI models by 18% in prediction accuracy, validated through backtesting on 50,000+ historical stock records dating back over a decade.
